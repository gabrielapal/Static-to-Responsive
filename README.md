# Static to Responsive Website

Skillcrush coding responsive websites practice- transforming a website so that it uses breakpoints and flexbox to be fully responsive.

### Skills practiced

- Flexbox
  - display
  - justify-content
  - flex-flow
  - max-width
  - width
- Typography
  - ems
  - rems
- Chrome DevTools
- Breakpoints
